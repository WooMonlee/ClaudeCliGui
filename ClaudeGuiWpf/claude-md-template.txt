# Claude Code 全局行为规则（所有项目生效）

## 1. 语言
- 所有回复、解释、选项、提示、错误信息使用中文
- 思考过程优先使用中文

## 2. 遥测与隐私
- 确认遥测已关闭（disableTelemetry: true）
- 所有数据仅通过已配置的 API 端点路由，不向第三方发送额外信息

## 3. 镜像源
- npm registry 使用 https://registry.npmmirror.com（国内镜像）
- 其他包管理器同理优先使用国内源

## 4. 插件
- 检查 superpowers 插件是否已安装并启用
- 如未安装，自动执行安装命令并启用
- 其他插件安装前说明来源和功能，征得用户同意

## 5. 全局 vs 项目隔离
- 全局配置对所有项目生效
- 项目本地 .claude/ 目录中的设置仅对该项目有效
- 切换项目时自动加载该项目的 CLAUDE.md 和记忆

## 6. 主动反问
- 当需求存在模糊、矛盾或缺失关键信息时，及时反问
- 聚焦影响较大的因素：安全、数据持久性、兼容性

## 7. 项目独立性
- 每个项目使用独立文件夹
- 所有记忆保存在项目各自的 .claude/memory/ 中，不使用全局记忆目录
- 如项目目录中已有记忆文件，视为重装系统残留，应复用而非覆盖
- 目的：避免重装系统时丢失记忆数据

## 8. 网络
- 所有网络请求前自动检查系统 IE 代理设置
- 如代理可用则使用，否则直接访问
